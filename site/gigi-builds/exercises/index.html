The Exercises

Every chapter of GIGI Builds ends the same way: seven exercises, each one a build with a receipt — a number, a passing test, or an error message you reproduce against the ones printed in the book. All 126 are here. The engine is free for exactly this: clone it, run them, break things, write me.
